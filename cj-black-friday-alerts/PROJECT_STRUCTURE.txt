QUICK REFERENCE
===============

Installation:
  pip install -r requirements.txt
  python verify_installation.py

Setup:
  python init_products.py        # Add 70 products
  python tracker.py              # First price check
  python app.py                  # Start dashboard

Access:
  Dashboard:  http://localhost:5000
  API:        http://localhost:5000/api/health

Automation:
  chmod +x scheduler.sh
  crontab -e
  # Add: 0 8 * * * /full/path/to/scheduler.sh

Testing:
  python test_scraper.py         # Test scraper
  curl http://localhost:5000/api/stats  # Test API
