# Related Project Management Commands

## Planning and Strategy
- `/create-roadmap` - Create project roadmap from specs and goals
- `/suggest-next-task` - AI-driven task prioritization based on telemetry
- `/generate-spec` - Create specification from requirements

## Analysis and Metrics
- `/analyze-velocity` - Analyze development velocity from telemetry
- `/load-telemetry` - Load and analyze workflow execution data

## Context Management
- `/load-spec` - Load specification for context
- `/load-workflow` - Load workflow definition
- `/explain-codebase` - Generate codebase overview and architecture
